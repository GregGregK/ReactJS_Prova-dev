## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/GregGregK/ReactJS_Prova-dev.git
```

Entre no diretório do projeto

```bash
  cd ReactJS_Prova-dev
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
